# Logo Placement

Place your JLD Minerals logo file here as `jld-logo.png`.

**Supported formats:**
- PNG (recommended)
- JPG/JPEG
- SVG

**Recommended size:**
- Header logo: 40px height (width will auto-scale)
- Login page logo: 80px height

**File name:** `jld-logo.png`

If the logo file is not found, the system will automatically fall back to displaying a diamond icon.
